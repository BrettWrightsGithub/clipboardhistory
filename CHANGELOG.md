# Changelog

## [1.0.0] - 2025-05-03
### Features
- Menu bar clipboard history manager
- Global hotkey (Option+V) to toggle history
- Search through clipboard entries
- Pin important entries
- Auto-paste: select an entry to paste directly into the active app
- Configurable max items and retention period
- Launch at Login support
- Core Data persistence with SQLite backend
- Keyboard navigation (arrow keys + Enter)
